Several wrappers and interfaces exist to use pandoc in other programming languages or via a graphical user interface (**GUI**). A **wrappers** can be used to run the pandoc executable while an **interface** give access to the abstract syntax tree (AST) of Pandoc documents, for instance to create filters. A **generator** is an interface that can only be used to create pandoc documents but not to modify them.
